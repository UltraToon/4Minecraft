## Temporary Setup (USE THIS)
Create a new shortcut
Add shell script module and go through preferences
Paste this in:
```sh
#!/bin/bash
bash <(curl -fsSL https://raw.githubusercontent.com/UltraToon/4Minecraft/refs/heads/main/setup.sh)
```
Press run in the top right corner, or macos menu bar in the shortcuts icon, or on the shortcuts homepage

## How to setup:
- Go to [**releases**](https://github.com/UltraToon/4Minecraft/releases/tag/Release)
- Read instructions
